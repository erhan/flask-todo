Flask Todo Uygulaması

Kurulum :

  • Github'tan clone edin : $ git clone https://github.com/erhan/flask-todo
  • Virtualenv klasörünü oluşturun : $ virtualenv -p python3.5 env
  • Gerekli modülleri `requirements.txt` dosyasından kurun. : $ env/bin/pip install -r requirements.txt
  • Veritabanını güncelleyin.(Default kullanıcı oluşturur.) : $ env/bin/python migrate.py db upgrade
  • Uygulamayı çalıştırın : $ env/bin/python run.py

Kaynak Kodlar : https://github.com/erhan/flask-todo
